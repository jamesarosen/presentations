JSHint: Multiple Versions

You work on many projects; each prefers a specific version of JSHint

Install Locally

  "devDependencies": {
    "jshint": "~1.1.0"
  }
}
npm install

Ignore warnings about -g

Run Locally
node_modules/jshint/bin/hint myFile.js
